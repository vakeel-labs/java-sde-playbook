# Searching Algorithms

Binary search is one of the most misunderstood topics — it appears in far more situations than just sorted array lookups.

## Topics Covered

- Linear search
- Binary search (classic)
- Binary search on rotated sorted array
- Binary search on answer (search space)
- Finding first/last occurrence
- Search in 2D sorted matrix
- Peak element finding
- Exponential search
- Interpolation search
- Ternary search

## Key Insight: Binary Search on Answer

Many problems that say "find minimum/maximum X such that condition holds" can be solved with binary search on the answer space, not an array.
