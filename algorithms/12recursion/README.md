# Recursion

Recursion is the backbone of tree traversal, divide and conquer, backtracking, and dynamic programming. Building strong recursive intuition is foundational.

## Topics Covered

- Base case and recursive case design
- Call stack and stack frame understanding
- Head recursion vs tail recursion
- Divide and conquer strategy
- Merge sort and quick sort recursion
- Tower of Hanoi
- Recursive string and array problems
- Converting recursion to iteration (with explicit stack)
- Memoization as an enhancement to recursion
- Mutual recursion

## Key Concepts

- "Trust the recursion" mental model
- Recurrence relation derivation
- Master theorem for time complexity
- Recursive vs iterative trade-offs (stack overflow risk)

## Practice Problems

- Factorial and Fibonacci
- Tower of Hanoi
- Flatten Nested List
- Power Function (fast exponentiation)
- Generate Parentheses
- Decode String
