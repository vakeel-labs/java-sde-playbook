# Stack & Queue

Stacks and queues are fundamental structures used in expression parsing, BFS/DFS, and many greedy problems.

## Topics Covered

- Stack: push, pop, peek, isEmpty
- Queue: enqueue, dequeue, circular queue
- Monotonic stack (increasing and decreasing)
- Monotonic deque (sliding window maximum)
- Stack-based expression evaluation (infix, postfix, prefix)
- BFS using a queue
- Priority Queue (Min/Max Heap)
- Deque (Double-ended queue)
- Implement Stack using Queues and vice versa

## Common Interview Patterns

- Monotonic stack for next greater/smaller element
- Queue for level-order traversal (BFS)
- Stack for backtracking / undo operations
