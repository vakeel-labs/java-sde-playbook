# Math & Number Theory

Mathematical foundations appear in many interview problems — from prime sieves to modular arithmetic used in large number computations.

## Topics Covered

- GCD and LCM (Euclidean algorithm)
- Prime numbers and Sieve of Eratosthenes
- Modular arithmetic and modular exponentiation
- Factorial and combinatorics (nCr, nPr)
- Fast exponentiation (binary exponentiation)
- Integer overflow handling
- Fibonacci with matrix exponentiation
- Catalan numbers
- Pigeonhole principle
- Number base conversions

## Common Interview Patterns

- Use modular arithmetic to avoid overflow: (a * b) % MOD
- GCD for simplifying fractions
- Sieve for prime-related problems
- Count divisors using prime factorization

## Practice Problems

- Count Primes (Sieve of Eratosthenes)
- Power(x, n) — fast exponentiation
- Ugly Number I & II
- Excel Sheet Column Number (base conversion)
- Happy Number
- Reverse Integer
