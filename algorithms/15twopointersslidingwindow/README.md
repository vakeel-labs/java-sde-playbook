# Two Pointers & Sliding Window

Two of the most versatile patterns in interviews — applicable to arrays, strings, and linked lists to reduce O(n²) brute force to O(n).

## Topics Covered

### Two Pointers
- Opposite-direction pointers (sorted arrays, palindromes)
- Same-direction pointers (fast/slow, partition)
- Three-pointer approach (3Sum, Dutch Flag)
- Linked list two-pointer patterns

### Sliding Window
- Fixed-size window (max sum subarray of size k)
- Variable-size window (smallest subarray with sum >= target)
- Window with constraint (at most k distinct characters)
- Sliding window maximum (monotonic deque)

## When to Use

- "Contiguous subarray/substring" → Sliding window
- "Pair with target sum in sorted array" → Two pointers
- "Remove duplicates in-place" → Two pointers
