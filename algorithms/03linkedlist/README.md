# Linked List

Linked lists test pointer manipulation and are the basis for many advanced data structures (stacks, queues, LRU cache).

## Topics Covered

- Singly and doubly linked lists
- Traversal and node insertion/deletion
- Reversal (iterative and recursive)
- Fast and slow pointer technique (Floyd's cycle detection)
- Merge two sorted lists
- Finding the middle of a linked list
- Detecting and removing a cycle
- Intersection of two linked lists
- Reverse in k-groups
- LRU Cache implementation

## Common Interview Patterns

- Two-pointer (fast/slow) for cycle and middle
- Dummy node technique to simplify edge cases
- Recursive vs iterative reversal trade-offs

## Practice Problems

- Reverse a Linked List
- Detect Cycle (Floyd's Algorithm)
- Merge Two Sorted Lists
- Remove Nth Node From End
- Copy List with Random Pointer
- LRU Cache
- Flatten a Multilevel Doubly Linked List
