# SQL Basics — DDL & DML

The foundational SQL commands every developer must know. Covers creating and modifying schema (DDL) and manipulating data (DML).

## DDL — Data Definition Language

- CREATE TABLE with data types and constraints
- ALTER TABLE (add, modify, drop columns)
- DROP TABLE and TRUNCATE
- Primary Key, Foreign Key, UNIQUE, NOT NULL, CHECK constraints
- DEFAULT values and auto-increment (IDENTITY / SERIAL)

## DML — Data Manipulation Language

- SELECT with WHERE, ORDER BY, LIMIT/OFFSET
- INSERT INTO (single row and bulk insert)
- UPDATE with WHERE clause
- DELETE with WHERE clause
- DISTINCT and elimination of duplicates

## DQL — Data Query Language

- Filtering with WHERE (AND, OR, NOT, BETWEEN, IN, LIKE)
- NULL handling (IS NULL, IS NOT NULL, COALESCE)
- Aliases (column and table aliases)
- CASE WHEN expressions
- String, numeric, and date functions
