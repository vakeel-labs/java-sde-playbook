# Easy Database Interview Questions

Foundational SQL and database concept questions. Expected to be answered quickly and confidently in any technical interview.

## SQL Query Topics

- SELECT with WHERE and ORDER BY
- Basic INNER JOIN between two tables
- COUNT, SUM, AVG with GROUP BY
- Finding NULL values
- Removing duplicates with DISTINCT
- Basic LIKE pattern matching

## Conceptual Topics

- What is a primary key vs foreign key?
- What is normalization and why is it used?
- What is the difference between DELETE, TRUNCATE, and DROP?
- What is a JOIN and what are the types?
- What is an index and why do we use it?
- What is the difference between WHERE and HAVING?

## Sample Questions

1. Find all employees with salary greater than 50,000
2. List departments with more than 5 employees
3. Find the second highest salary in the employee table
4. Get all customers who have placed at least one order
5. Find duplicate email addresses in a users table
