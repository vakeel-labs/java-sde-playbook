# Strings

String problems are extremely common in interviews and often combine multiple techniques: hashing, two pointers, and dynamic programming.

## Topics Covered

- String traversal and character manipulation
- Substring vs subsequence concepts
- Palindrome detection and expansion
- Anagram and permutation checks
- Sliding window on strings
- KMP algorithm (pattern matching)
- Rabin-Karp algorithm
- Z-algorithm
- String compression and encoding
- Parsing and tokenization

## Common Interview Patterns

- HashMap for character frequency
- Two-pointer for palindromes
- Sliding window for substrings with constraints
- Dynamic programming for edit distance / LCS

## Practice Problems

- Longest Palindromic Substring
- Minimum Window Substring
- Group Anagrams
- Longest Common Subsequence
- Edit Distance
- Valid Parentheses
- String to Integer (atoi)
