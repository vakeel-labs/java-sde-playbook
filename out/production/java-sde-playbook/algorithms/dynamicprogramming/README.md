# Dynamic Programming

DP is arguably the hardest DSA topic and the most frequently tested at top companies. The key is recognizing overlapping subproblems and optimal substructure.

## Topics Covered

- Memoization (top-down DP)
- Tabulation (bottom-up DP)
- 1D DP: Fibonacci, climbing stairs, house robber
- 2D DP: grid paths, edit distance, LCS
- Knapsack problems: 0/1 knapsack, unbounded knapsack
- Subsequence problems: LCS, LIS, LCSS
- Partition problems: subset sum, equal partition
- String DP: palindrome, edit distance, wildcard matching
- Interval DP: matrix chain multiplication, burst balloons
- Digit DP
- Bitmask DP
- DP on trees

## Common Patterns to Recognize

- "Count ways to..." → DP
- "Minimum/maximum cost to..." → DP
- "Is it possible to..." → DP (often knapsack variant)
