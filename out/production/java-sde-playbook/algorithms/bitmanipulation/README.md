# Bit Manipulation

Bit manipulation enables constant-time solutions to problems that would otherwise require loops. Frequently appears in competitive programming and system-level interviews.

## Topics Covered

- Binary representation of integers
- Bitwise operators: AND (&), OR (|), XOR (^), NOT (~)
- Left shift (<<) and right shift (>>)
- Check, set, clear, toggle a bit
- Count set bits (Brian Kernighan's algorithm)
- Power of 2 check
- XOR tricks (find missing number, single number)
- Bit masking and bitmask DP
- Gray code
- Swap without temp variable

## Common XOR Properties

- a ^ a = 0
- a ^ 0 = a
- XOR is commutative and associative

## Practice Problems

- Single Number (I, II, III)
- Number of 1 Bits (Hamming Weight)
- Reverse Bits
- Missing Number
- Sum of Two Integers (without + operator)
- Subsets using bitmask
