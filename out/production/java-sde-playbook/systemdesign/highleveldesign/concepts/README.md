| [Database Sharding & Partitioning](./database-sharding-partitioning) | How do we scale databases horizontally? |
| [Consistent Hashing](./consistent-hashing) | How do we distribute data without massive rebalancing? |

| [SQL vs NoSQL](./sql-vs-nosql) | When do we use relational vs non-relational databases? |
| [Replication & Consistency](./replication-and-consistency) | How do we keep data safe and consistent across nodes? |
| [API Design (REST & gRPC)](./api-design-rest-grpc) | How do services expose and call each other? |
| [Event-Driven Architecture](./event-driven-architecture) | How do we build systems with async event flows? |
