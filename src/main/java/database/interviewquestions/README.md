# Database Interview Questions

Curated database interview questions organized by difficulty. Covers SQL query writing, database concepts, design, and optimization.

## Contents

- [Easy](./easy) — Fundamentals: basic queries, joins, aggregations
- [Medium](./medium) — Window functions, subqueries, indexing, transactions
- [Hard](./hard) — Optimization, design trade-offs, advanced concepts

## Question Categories

- SQL query writing (given a schema, write a query)
- Conceptual questions (explain ACID, isolation levels, indexing)
- Database design (design a schema for a given system)
- Performance and optimization
- NoSQL vs SQL trade-off questions
- Specific DB internals (how does a B-tree index work?)
