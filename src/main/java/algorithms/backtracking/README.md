# Backtracking

Backtracking is a systematic method to explore all potential solutions by building candidates incrementally and abandoning them when they fail constraints.

## Topics Covered

- Backtracking template (choose, explore, unchoose)
- Subsets (power set generation)
- Permutations (with and without duplicates)
- Combinations (k-combinations, combination sum)
- N-Queens problem
- Sudoku solver
- Word search in a grid
- Palindrome partitioning
- Letter combinations of phone number
- Generate all valid parentheses

## Key Concepts

- State space tree visualization
- Pruning to reduce unnecessary exploration
- Difference between subsets, permutations, and combinations
