# Tries (Prefix Trees)

A trie is a tree-like data structure specialized for string prefix operations — the backbone of autocomplete, spell-checkers, and IP routing.

## Topics Covered

- Trie node structure design
- Insert, search, and startsWith operations
- Word search in a grid using Trie
- Autocomplete / typeahead system
- Longest common prefix using Trie
- Counting words with a given prefix
- Delete operation in Trie
- Compressed Trie (Radix Tree)
- Bit Trie (for XOR maximization)

## Complexity

- Insert/Search/StartsWith: O(m) where m = word length
- Space: O(ALPHABET_SIZE * m * n)
