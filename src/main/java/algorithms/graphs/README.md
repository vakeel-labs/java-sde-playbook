# Graphs

Graphs model relationships between entities and appear in routing, social networks, and scheduling problems. Master BFS/DFS and their applications.

## Topics Covered

- Graph representations: adjacency list, adjacency matrix, edge list
- BFS (Breadth-First Search) and applications
- DFS (Depth-First Search) and applications
- Cycle detection (directed and undirected)
- Topological sort (Kahn's algorithm + DFS)
- Dijkstra's algorithm (shortest path, weighted)
- Bellman-Ford algorithm (negative weights)
- Floyd-Warshall (all-pairs shortest path)
- Minimum Spanning Tree: Prim's and Kruskal's
- Union-Find (Disjoint Set Union)
- Strongly Connected Components (Kosaraju's, Tarjan's)
- Bipartite graph check
- Island problems (grid as graph)

## Common Interview Patterns

- BFS for shortest path in unweighted graphs
- DFS for connected components and cycle detection
- Topological sort for dependency ordering
