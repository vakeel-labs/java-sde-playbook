# Heap / Priority Queue

Heaps are essential for problems involving "top K", "kth largest/smallest", and streaming data where you need efficient min/max access.

## Topics Covered

- Binary heap structure (min-heap and max-heap)
- Heapify operation and heap sort
- Java PriorityQueue class and custom comparators
- Top K elements pattern
- K-way merge pattern
- Two-heap pattern (median of data stream)
- Sliding window with heap
- Task scheduling with heaps

## Key Properties

- Insert: O(log n)
- Delete min/max: O(log n)
- Peek min/max: O(1)
- Build heap: O(n)
