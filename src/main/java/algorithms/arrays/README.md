# Arrays

The foundation of DSA. Arrays underlie nearly every other data structure and appear in the majority of coding interview problems.

## Topics Covered

- Array traversal and manipulation
- Prefix sum and difference arrays
- Two pointer technique (opposite ends, same direction)
- Sliding window (fixed size and variable size)
- Kadane's algorithm (maximum subarray)
- Sorting-based array problems
- Dutch National Flag problem
- Rotate array / matrix
- Merge intervals
- Matrix (2D array) traversal

## Common Interview Patterns

- Subarray problems (sum, product, length)
- In-place modifications
- Hash map + array combinations
- Order statistics (k-th largest/smallest)

## Practice Problems

- Two Sum, Three Sum, Four Sum
- Maximum Subarray (Kadane)
- Merge Intervals
- Product of Array Except Self
- Trapping Rain Water
- Jump Game I & II
