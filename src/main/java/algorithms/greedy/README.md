# Greedy Algorithms

Greedy algorithms build solutions piece by piece, always choosing the locally optimal option. The challenge is proving that local optima lead to a global optimum.

## Topics Covered

- Activity selection / interval scheduling
- Fractional knapsack
- Huffman encoding
- Job sequencing with deadlines
- Gas station problem
- Jump game
- Minimum number of platforms
- Task scheduler
- Meeting rooms problems

## Key Concepts

- When does greedy work? (Exchange argument / matroid theory)
- Greedy vs DP decision
- Sorting as a prerequisite for greedy
